# 🎬 Netflix Clone - Imersão Alura

Projeto desenvolvido durante a Imersão Alura com o objetivo de recriar a interface da Netflix, aplicando conceitos de HTML, CSS e JavaScript.

---

## 📌 Sobre o Projeto

Este projeto simula a experiência inicial da Netflix, permitindo:

- Seleção de perfil de usuário
- Persistência de dados com `localStorage`
- Redirecionamento entre páginas
- Exibição dinâmica do nome e foto do perfil no catálogo
- Alternância entre modo claro e escuro 🌙☀️

---

## 🧠 Conceitos Aplicados

- Estruturação semântica com HTML5
- Estilização moderna com CSS3 (Flexbox)
- Manipulação de DOM com JavaScript
- Armazenamento local (`localStorage`)
- Navegação entre páginas
- Organização de código em múltiplos arquivos

---

## 🎯 Funcionalidades

✔ Seleção de perfil  
✔ Armazenamento do perfil selecionado  
✔ Redirecionamento para o catálogo  
✔ Atualização dinâmica do header (nome + imagem)  
✔ Tema Dark/Light persistente  
✔ Layout inspirado na Netflix
